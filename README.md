# Blog Agent

A lightweight agent for generating, organizing, and publishing blog content. Designed to be modular and easy to extend for different content sources, AI models, and publishing targets.

## Features
- Generate draft blog posts from the given topic

## Getting started

1. Clone the repository
```
git clone <repo-url>
cd Blog-Agent
```

2. Install dependencies
- If Python:
```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

3. Configure
Create a `.env` file or set environment variables required by connectors (examples):
```
GROQ_API_KEY=your_key_here

```
